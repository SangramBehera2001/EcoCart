# 🛒 EcoCart – Sustainable Shopping Platform 🌱

**EcoCart** is a full-stack e-commerce web application focused on promoting sustainable shopping by showcasing only eco-friendly products. It features a Spring Boot backend with a frontend built using HTML, CSS, JavaScript, and Bootstrap.

---

## 🚀 Tech Stack

### 🧠 Backend:
- Java 17+
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL 
- Lombok

### 💻 Frontend:
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

### 🔧 Tools:
- IntelliJ IDEA (Backend)
- IntelliJ (Frontend)
- Git + GitHub
- Postman (API testing)

---

## 🌿 Key Features

- ✅ Display only **eco-friendly products**
- ♻️ **Carbon footprint meter** for each product
- 🔍 Search and filter products by eco-friendly status
- 🛍️ Static UI pages for product listing, cart, and more
- 🔐 User authentication (planned)
- 📦 Admin dashboard for product management (coming soon)
- 📊 Eco score system (planned)
